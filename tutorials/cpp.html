The classic first C++ program demonstrates basic structure, compilation, and output operations. Understanding this example ensures your environment is set up correctly.

Variables & Data Types

Learn how to declare variables and use data types like int, double, char, and bool. Here's how you get started: 


// Declaring variables in C++
#include 
using namespace std;

int main() {
  int age = 25;
  double height = 5.9;
  char grade = 'A';
  bool isStudent = true;

  cout << "Age: " << age << endl;
  cout << "Height: " << height << endl;
  cout << "Grade: " << grade << endl;
  cout << "Student: " << (isStudent ? "Yes" : "No") << endl;
  return 0;
}

Functions let you segment your code into reusable units. You can pass parameters and return values.

Pointers & Memory

Pointers store memory addresses. Allocating with new and releasing with delete gives manual control over memory. 


int* ptr = new int(42);
std::cout << *ptr << std::endl;
delete ptr; // free the memory

Object-Oriented Programming

OOP adds structure through classes and objects, supporting encapsulation and reusability. 


class Student {
public:
  Student(std::string n, int a) : name(n), age(a) {}
  void info() const {
    std::cout << name << ", " << age << " years old" << std::endl;
  }
private:
  std::string name;
  int age;
};

int main() {
  Student s("Dev", 3);
  s.info();
  return 0;
}

Exception Handling & File I/O

Handle errors cleanly with try / catch. Read/write files using fstream. 


#include 

try {
  std::ifstream file("data.txt");
  if (!file) throw "Cannot open file!";
  std::string line;
  while (std::getline(file, line)) std::cout << line << std::endl;
} catch (const char* msg) {
  std::cerr << "Error: " << msg << std::endl;
}

Smart Pointers & RAII

Smart pointers such as std::unique_ptr, std::shared_ptr, and std::weak_ptr manage dynamic memory using RAII (Resource Acquisition Is Initialization), preventing memory leaks and dangling pointers. ( GeeksforGeeks: Smart Pointers , cppreference: Memory Management )

C++20 Features & Coroutines

C++20 introduces features like coroutines (for asynchronous routines), modules, concepts, and enhanced constexpr. ( cppreference: C++20 ) 

#include <coroutine>
#include <iostream>
#include <optional>

template<typename T>
struct Generator {
  struct promise_type {
    std::optional<T> current_value;
    Generator get_return_object();
    std::suspend_always initial_suspend() noexcept { return {}; }
    std::suspend_always final_suspend() noexcept { return {}; }
    std::suspend_always yield_value(T value) {
      current_value = value;
      return {};
    }
    void return_void() {}
    void unhandled_exception() { std::terminate(); }
  };

  std::coroutine_handle<promise_type> handle;
  ~Generator() { if (handle) handle.destroy(); }
  std::optional<T> next() {
    if (!handle.done()) {
      handle.resume();
      return handle.promise().current_value;
    }
    return std::nullopt;
  }
};

Generator<int> generate() {
  for (int i = 1; i <= 3; ++i) co_yield i;
}

int main() {
  auto gen = generate();
  while (auto val = gen.next()) std::cout << *val << std::endl;
  return 0;
}
